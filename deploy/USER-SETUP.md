# 랜선어항 설치 안내 (동료용)

Teams에서 "랜선어항"을 쓰려면 아래 **2단계**를 한 번만 해주세요.
`fishtank.formationlabs.co.kr` 은 공인 도메인으로 등록되어 있어 별도 주소
등록 작업이 필요 없습니다.

준비물: 담당자에게 받은 파일 1개
- `fishtank-teams.zip` (Teams 앱 패키지)

> 참고: 예전 사내 전용 도메인(`fishtank.fllab.internal`) 배포 때 쓰던
> `add-hosts.bat`/`remove-hosts.bat` 는 현재 배포(공인 도메인)에서는
> **사용하지 않습니다**. 삭제하지 않고 남겨둔 참고용 파일입니다.

---

## 1단계 — Teams에 앱 추가

1. Teams 실행 → 왼쪽 아래 **앱** 클릭
2. **앱 관리** → **앱 업로드** → **사용자 지정 앱 업로드**
3. 받은 `fishtank-teams.zip` 선택 → 추가

> "사용자 지정 앱 업로드" 항목이 안 보이면 회사 정책상 막힌 것입니다.
> 그 경우 담당자에게 문의하세요.

## 2단계 — 접속 확인

1. Teams에서 방금 추가한 **랜선어항** 앱 열기
2. 잠시 후 물고기 어항 화면이 나오면 완료입니다

> **"연결이 거부되었습니다" 또는 인증서 경고**가 뜨면:
> 브라우저에서 `https://fishtank.formationlabs.co.kr` 접속해 자물쇠(정상)인지 확인하세요.
> 경고가 뜨면 인증서 신뢰가 안 된 것이라 담당자에게 알려주세요.

---

## 제거하려면

Teams 앱은 앱 관리에서 삭제하면 됩니다.

## 자주 묻는 것

| 증상 | 해결 |
|------|------|
| "사용자 지정 앱 업로드" 안 보임 | 회사 정책 — 담당자 문의 |
| 앱은 열리는데 "연결 거부" | 담당자에게 DNS 등록 여부 확인 요청 |
| 인증서 경고 | 담당자에게 인증서 상태 확인 요청 |
| 로그인 실패 | Teams 안에서 열었는지 확인(브라우저 직접 접속은 안 됨) |
